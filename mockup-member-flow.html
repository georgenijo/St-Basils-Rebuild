Member Dashboard — What they see
Overview Tab
At-a-glance family status
Shows
  • Next due date + amount
  • Family members count
  • Active shares count for this year
  • Outstanding balance
  • Recent activity feed (all types)
Membership Tab
Plan status + dues history
Shows
  • Plan status (active / expired / pending)
  • Plan type (monthly $100 / annual $1,000)
  • Next payment + paid-through date
  • Year progress bar
  • Dues payment history (membership only)
  • Annual summary (paid vs remaining)
Family Tab
Manage household members
Actions MEMBER
  • View family members (name, relationship)
  • Add / remove family members
  • Edit family details (phone, address)
  • Head of household is the account owner
Payments Tab
Events, donations, shares — not dues
Shows (excludes membership dues)
  • Event costs assigned to your family
  • Donations (general, car blessing, caroling, etc.)
  • Share purchases
  • Payment history with dates + amounts
  • Outstanding balance (non-membership)
Record a Donation
Type, amount, optional note
Donation types
  • General donation (any amount)
  • Car blessing donation
  • Christmas caroling donation
  • Event-specific donation
  • Other (free text)

v2 — Stripe / payment processor

Shares Tab
Names remembered in services
Shows
  • Shares purchased this year
  • Each share: person's name + $50
  • Total spent on shares
  • Previous years' shares (history)
Buy Shares
Add names, $50 each
Flow
  • Enter person's name
  • Add multiple names at once
  • Shows running total ($50 x count)
  • Submit → admin sees in their dashboard
  • v1: payment tracked, not processed online
Admin Side — Managing member data
ADMIN ONLY
Assign Event Costs
After an event, assign split to families
Flow
  • Select event from list
  • Enter total cost
  • Select which families attended
  • Auto-splits or manual amounts
  • Shows up in member's Payments tab
ADMIN ONLY
Record Payments Received
Mark dues, donations, shares as paid
Flow
  • Select family / member
  • Select payment type (dues, share, event, donation)
  • Enter amount + method (cash, check, Zelle)
  • Updates member's balance + history
  • Audit logged
ADMIN ONLY
Manage Shares List
View all shares for services
Shows
  • All names to remember this year
  • Which member bought each share
  • Payment status per share
  • Export list for service use
  • Year filter
